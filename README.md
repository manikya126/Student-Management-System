# Student-Management-System
## Project Description
A console-based Java application that helps manage student records. It allows users to add, update, search, delete, and display student information.
## Features
- Add student records
- Update exixting student details
- Search for a student by ID
- Delete student records
- Display all student records
## Technologies Used
- Java (Core Java, OOPs concepts)
## How to Run the Project
1. Ensure you have Java installed (JDK 8 or above).
2. Open the project folder in VS Code.
3. Compile and run `Main.java` to start the application.
